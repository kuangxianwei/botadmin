# BotAdmin
## BotAdmin 安装版
## 当前版本为: v1.9.3
## 安装环境: centos 7.x
### 安装前最好先升级系统软件 yum -y upgrade

## 安装方法:
### wget http://auth.nfivf.com:9090/mirror/install.sh && sh ./install.sh
### 参数 mirror=cn 则下载国内的镜像
#### 例如:wget http://auth.nfivf.com:9090/mirror/install.sh && sh ./install.sh mirror=cn
#### 直接指定 sh ./install.sh mirror=https://github.com/kuangxianwei/botadmin/archive/master.zip
#### 默认是Github下载
