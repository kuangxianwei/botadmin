# BotAdmin
## BotAdmin 安装版
# 当前版本为: v1.8.5
## 安装方法:
### wget http://auth.nfivf.com:9090/mirror/install.sh && sh ./install.sh
### 参数 mirror=cn 则下载国内的镜像
### 直接指定 mirror=https://github.com/kuangxianwei/botadmin/archive/master.zip
### 默认是Github下载
### ngx_filter=off
### 默认NGINX启用第三方过滤替换模块 ngx_filter=off 为不启用
